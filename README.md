# REACT SHOP API

This React - Front End project is based on a multi-product online shop using an API.

## Technologies:
- Frontend: ReactJS, TypeScript, CSS, SCSS, HTML

## API LINK:
- https://api.escuelajs.co/api/v1/products


Execute the command: 

```bash
npm i
npm run start
```

## Usage

Go to http://localhost:3000/
